## Правила

1. Каждый пункт должен быть конкретным действием
2. Укажи срок / дедлайн для каждого шага (если применимо)
3. Укажи ответственного (кто выполняет: юрист, бухгалтер, директор)
4. Ссылайся на применимые нормы (ФЗ «Об ООО», ГК РФ, НК РФ)
5. Отметь шаги, связанные с государственной регистрацией

## Формат

Нумерованный чек-лист:
1. [ ] Действие — срок — ответственный — основание
